Key Visual Effects

🌟 Bloom + Glow

UnrealBloom post-processing. Every node and connection glows with neon halos. Intensity pulses with "heartbeat" animation.

⚡ Energy Streams

Particles flow along connection lines between nodes like electric current. Speed increases when user hovers near a connection.

🎮 Boot Sequence

Page loads with fake terminal boot: system checks, "NEURAL MAP LOADING...", glitch effects, then camera bursts into the 3D scene.

🔊 Audio Layer

Ambient synth hum + click/hover sound effects. Subtle, sci-fi, non-intrusive. User can toggle audio on/off.
